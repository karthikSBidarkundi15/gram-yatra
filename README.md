# SmartBusTrack

## Overview

SmartBusTrack is a real-time public transport tracking system designed for small cities. The project aims to improve public transportation accessibility by providing live bus tracking, estimated arrival times (ETA), and route monitoring using GPS and IoT technologies.

The system helps commuters plan their journeys efficiently while enabling transport authorities to monitor vehicle movement in real time.

---

## Features

* Real-time bus location tracking
* GPS-based vehicle monitoring
* Estimated Time of Arrival (ETA)
* Route tracking and management
* Web and mobile accessibility
* SMS-based updates for low-connectivity regions
